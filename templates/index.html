欢迎使用取证仪表盘

请按照以下步骤完成安装并开始使用取证仪表盘：

  • 步骤1：下载最新版本安装包并解压到本地目录。
  • 步骤2：安装依赖环境（Python）。
  • 步骤3：执行 pip install -r requirements.txt 安装 Python 依赖。
  • 步骤4：运行 python app.py 自动启动系统，或浏览器访问 http://localhost:5000。
  • 步骤5：使用侧边栏功能进行关键字搜索、以图搜图或即时取证以及定时取证功能。
